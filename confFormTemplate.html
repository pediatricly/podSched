This El Nino program calculates conference attendance based on residents' schedules.

This program tallies attendance using resident schedules.
It does NOT read the Amion schedule directly! It uses data saved by the El Nino "Updater" (aka blockParse). Obviously, if those data are not current, you will get inaccurate results and residents will be pissed. So, after any substantive schedule change, it is critical to update El Nino before using this program.
El Nino was last updated on $updated (yyyy-mm-dd).
If the schedule might have changed since then, go to El Nino Updater to update El Nino's schedule data.
